# 🚀 Kanban Board - Frontend

Sistema Kanban desenvolvido como desafio técnico, consumindo uma API REST com autenticação JWT.

---

## 📌 Sobre

Aplicação web para gerenciamento de tarefas em fluxo Kanban, com suporte a permissões, histórico e validações de negócio.

---

## 🛠️ Tecnologias

- Next.js
- React
- TypeScript
- CSS Modules
- @hello-pangea/dnd
- JWT
- API REST (FastAPI + PostgreSQL)

---

## ⚙️ Funcionalidades

- Autenticação de usuários
- Listagem de boards
- Kanban com drag-and-drop
- CRUD de cards
- Comentários e histórico
- Controle de permissões (admin, editor, viewer)
- WIP limit por coluna
- Movimentação com observação obrigatória
